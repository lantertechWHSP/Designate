# WHSP (By Designate)

## Summary
    WHSP uses the following stack
    * React
    * Next
    * Vercel
    * DatoCMS

## Running code on your local machine
1) Run `npm install`
2) Run `npm run dev`
   * Note: ensure that you have the appropraite .env variables
3) Open localhost: `http://localhost:3000`
